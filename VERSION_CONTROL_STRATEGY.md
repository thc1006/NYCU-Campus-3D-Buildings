# Version Control Strategy / 版本控制策略

**TL;DR**: Main branch for releases, semantic versioning, GitHub Releases for large files.

**簡介**: main 分支用於發布，語義化版本，大檔案放 GitHub Releases。

## Branching / 分支策略

- main - Stable releases / 穩定發布
- feature/* - New features / 新功能
- fix/* - Bug fixes / 錯誤修復

## Versioning / 版本號

Format: vMAJOR.MINOR.PATCH (e.g., v1.0.0)

## Large Files / 大檔案

Use GitHub Releases (not in repository) / 使用 GitHub Releases（不在儲存庫中）
